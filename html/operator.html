연산자란?

+, -, *, / 순서대로 더하기, 빼기, 곱하기, 나누기를 c언어에서 표현하는 방법입니다. 우리가 기본적으로 제일 많이 사용하는 사칙연산이 결국 연산자라고 할 수 있으실 겁니다. 그리고 거기서 플러스로 어떤 수를 나눈 나머지를 출력하는 나머지 연산자 %도 있습니다.

          
#include <stdio.h>
#pragma warning (disable:4996)

int main()
{
    int a;
	  int b;
	
	  scanf("%d%d", &a, &b);
	  printf("%d + %d = %d", a, b, a+b);
	  printf("%d - %d = %d", a, b, a-b);
	  printf("%d * %d = %d", a, b, a*b);
	  printf("%d / %d = %d", a, b, a/b);
	  printf("%d % %d = %d, a, b, a%b);
	  return 0;
}

위의 소스 코드처럼 입력하시면 화면에 커서가 깜박거리고 있을 테니 숫자를 10 4 이렇게 입력해주시면 아래에 출력결과가 보이실 겁니다.

           

    10 + 4 = 14
    10 - 4 = 6
    10 * 4 = 40
    10 / 4 = 2
    10 % 4 = 2

이렇게 결과가 나오셨다면 우리는 간단한 사칙연산 프로그램을 만들어 본 것입니다. 참고로 위의 코드를 복사 붙여넣기 하지 마시고 직접 안 보고 써보시길 바랍니다. 그래야 실력이 늘어나는 것을 체감하실 수 있습니다.

연산자는 위의 사칙연산 뿐 아니라 = 도 연산자인데요. =은 대입 연산자라고 불리며 우선순위가 낮은 연산자입니다. 참고로 우선순위는 사칙연산과 =을 포함한 연산자인 산술연산자 > 관계 연산자 > 논리 연산자 순입니다.

위의 코드에서 int temp; 라는 변수를 선언한 후 printf("%d", temp = a+b); 라고 입력하시면 출력 결과가 4+6인 10이 출력됨을 알 수 있습니다. 따라서 산술 연산자인 +가 먼저 a의 값과 b의 값을 더한 뒤 정수형 변수 temp에 대입하여 %d를 통해 프로그램 창에 출력한 것을 알 수 있습니다.

복합대입 연산자

복합대입 연산자는 +=, -=, *=, /=, %= 등이 있습니다. +=은 더하고서 대입하라는 뜻입니다. 예를 들어 변수 a와 b가 있다고 했을 때 a에는 5, b에는 10이라는 값이 들어 있다고 했을 때 a += b;는 a = a + b; 와 같이 a와 b를 더한 15를 변수 a에 대입하라는 뜻입니다.

관계 연산자

관계 연산자는 기본적으로 참과 거짓을 가려내는 것입니다. 참(1)과 거짓(0) 중 하나만을 출력하는 것이죠. 예를 들어 정수형 변수 a에 10이, b에 5가 있다고 했을 때 a > b의 값은 1이 나온다고 생각하시면 됩니다.

관계 연산자의 종류로는 ==(같다), !=(같지 않다) >(크다), <(작다), >=(이상), <=(이하)가 있습니다.

논리 연산자

논리 연산자는 관계 연산자로 만들어진 조건들을 조합하여 연산하는 것입니다. 이것에는 AND와 OR, NOT이 있습니다.

&&(AND), ||(OR), !(NOT)은 참과 거짓을 이용한 연산이라고 할 수 있습니다. 예를 들어 (a >= 20) && (a <= 40)라는 연산이 있을 때 a가 만약 30이라면 그 연산의 결과는 참(1)일 것입니다. 하지만 a가 20보다 작거나 40보다 커져버리면 곧바로 그 연산의 결과는 거짓(0)이 됩니다.

AND는 둘 중 하나라도 0이면 0이 되는 연산이고 ||은 둘 중 하나라도 1이면 1이 되는 연산입니다. 그리고 NOT은 부정을 의미하면 만약 !(a >= 20)이라는 연산에서 a가 16이라서 a >= 20이라는 연산이 거짓(0)이 되면 !연산으로 인해 참(1)으로 바꾸는 즉, 거짓을 참으로, 참을 거짓으로 만드는 연산인 것입니다.

형변환 연산자와 sizeof() 연산자

형변환 연산자는 만약 int a; int b;라고 변수들을 선언한 후 a의 값이 8이고 b의 값이 3일 때 a / b 연산을 수행하여 2.6이라는 결과를 알고 싶다면 printf문에서 a와 b 둘 중 하나의 변수 앞에 (float)a라고 쓰고 %d를 %.1f로 바꾸면 2.6이라는 결과가 나올 것입니다.

sizeof()연산자는 괄호 때문에 함수로 보이지만 함수가 아닌 연산자라는 것을 명심해주시면 좋겠습니다. sizeof()의 괄호 안에 정수형 변수 a를 넣으면 a가 4바이트의 저장공간을 사용한다는 뜻에서 숫자 4를 출력합니다. printf("%d", sizeof(a));라고 쓰면 됩니다.
